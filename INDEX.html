üöÄ Quick Start Instructions

Get started in just a few minutes:

1. Install Dependencies

# Windows INSTALL.bat # macOS/Linux chmod +x INSTALL.sh ./INSTALL.sh

2. Configure Environment

Create backend/.env with MongoDB URI and JWT secret (see .env.example)

3. Start Backend

cd backend npm start # or npm run dev for development # Runs on http://localhost:5000

4. Start Frontend (new terminal)

cd frontend npm start # Runs on http://localhost:3000
